# ChatGPT-cli

## วิธีการติดตั้งและใช้งาน


_1. ดาวโหลด์_
```
git clone https://github.com/VarinCode/chatGPT-cli.git
```

_2. สร้าง environment_
```
py -m venv .venv
```

_3. เปิดใช้งาน_
```
.venv\scripts\activate
```

_4. ดาวโหลด์ libs_
```
pip install -r requirements.txt
```

_5. รัน_
```
py main.py
```
